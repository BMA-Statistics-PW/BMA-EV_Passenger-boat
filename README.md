## 📈 การวิเคราะห์ข้อมูล

### ข้อมูลสำคัญที่พบ

1. **ผู้โดยสารบ่ายมากกว่าเช้า**
   - บ่าย: 9,862 คน/เดือน
   - เช้า: 5,796 คน/เดือน
   - ส่วนแบ่ง: 62.8% เป็นผู้โดยสารบ่าย

2. **ยอดสูงสุดในช่วงเทศกาล**
   - มกราคม 2567 มีผู้โดยสารสูงสุด 24,657 คน
   - เป็นช่วงเทศกาลปีใหม่ที่มีการท่องเที่ยวเพิ่มขึ้น

3. **ความแตกต่างระหว่างปี**
   - ปี 2567 มีผู้โดยสารมากที่สุด (228,235 คน)
   - ปี 2568 ลดลงเล็กน้อย (201,231 คน)
   - ปี 2569 ยังมีข้อมูลเพียงบางส่วน

4. **ความสม่ำเสมอของบริการ**
   - เฉลี่ยรอบเรือ 612 รอบต่อเดือน
   - ความสม่ำเสมอในการให้บริการ

## 📝 ข้อมูลเพิ่มเติม

**ช่วงเวลาข้อมูล:** 15 มีนาคม 2566 - กุมภาพันธ์ 2569
**แหล่งข้อมูล:** 
- ข้อมูลจำนวนผู้โดยสารเรือไฟฟ้าคลองผดุงกรุงเกษม สำนักงานระบบขนส่ง สำนักการจราจรและขนส่ง
- วิเคราะห์โดย กลุ่มงานสถิติและวิจัย กองนโยบายและแผนงาน สำนักการจราจรและขนส่ง
 
**ความถี่อัปเดต:** ข้อมูลรายเดือน
